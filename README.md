# Study Planner

A modern study planner built for students to organize tasks, track progress and improve study habits.

## Live Demo

https://study-planner-eight-weld.vercel.app

## Features

- Task Management
- Study Sessions
- Progress Tracking
- Real-Time Sync
- Dark Mode
- User Accounts

## Tech Stack

- React
- Vite
- Firebase
- Firestore
- Tailwind CSS



## Future Plans

- Calendar Integration
